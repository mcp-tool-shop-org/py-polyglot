## About polyglot-gpu

polyglot-gpu is a Python port of [polyglot-mcp](https://github.com/mcp-tool-shop-org/polyglot-mcp). It translates text into 57 languages using Google's TranslateGemma model running locally through Ollama. Zero cloud dependency, zero API keys.

Use it as a **pip-installable Python library** in your own projects or as an **MCP server** for Claude Code, Claude Desktop, and other MCP clients.
