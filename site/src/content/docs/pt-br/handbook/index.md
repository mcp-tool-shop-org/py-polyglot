## Sobre o polyglot-gpu

polyglot-gpu é uma adaptação em Python do [polyglot-mcp](https://github.com/mcp-tool-shop-org/polyglot-mcp). Ele traduz texto para 57 idiomas usando o modelo TranslateGemma do Google, executado localmente através do Ollama. Sem dependência de nuvem, sem chaves de API.

Use-o como uma **biblioteca Python instalável via pip** em seus próprios projetos ou como um **servidor MCP** para Claude Code, Claude Desktop e outros clientes MCP.
