## À propos de polyglot-gpu

polyglot-gpu est une version Python de [polyglot-mcp](https://github.com/mcp-tool-shop-org/polyglot-mcp). Il traduit du texte dans 57 langues en utilisant le modèle TranslateGemma de Google, exécuté localement via Ollama. Aucune dépendance cloud, aucune clé API.

Utilisez-le comme une **bibliothèque Python installable via pip** dans vos propres projets, ou comme un **serveur MCP** pour Claude Code, Claude Desktop et d'autres clients MCP.
