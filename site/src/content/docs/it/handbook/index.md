## Informazioni su polyglot-gpu

polyglot-gpu è una versione in Python di [polyglot-mcp](https://github.com/mcp-tool-shop-org/polyglot-mcp). Traduce testi in 57 lingue utilizzando il modello TranslateGemma di Google, eseguito localmente tramite Ollama. Nessuna dipendenza dal cloud, nessuna chiave API.

Utilizzatelo come **libreria Python installabile tramite pip** nei vostri progetti o come **server MCP** per Claude Code, Claude Desktop e altri client MCP.
