## Acerca de polyglot-gpu

polyglot-gpu es una adaptación de Python de [polyglot-mcp](https://github.com/mcp-tool-shop-org/polyglot-mcp). Traduce texto a 57 idiomas utilizando el modelo TranslateGemma de Google, ejecutándose localmente a través de Ollama. No requiere conexión a la nube ni claves de API.

Utilícelo como una **biblioteca de Python instalable mediante pip** en sus propios proyectos o como un **servidor MCP** para Claude Code, Claude Desktop y otros clientes MCP.
